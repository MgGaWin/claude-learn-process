## ✨ 功能特性

| 功能 | 说明 |
|:---|:---|
| 🎬 **Remotion 视频** | 基于 React 的程序化视频生成 |
| 📝 **Claude Code 技能** | 自定义技能，扩展 Claude Code 能力 |
| 🎯 **TypeScript** | 全类型安全的开发体验 |
| 🎨 **动画系统** | 帧精确的动画控制 |

## 🚀 快速开始

### 安装

```bash
git clone https://github.com/MgGaWin/claude-learn-process.git
cd claude-learn-process
npm install
```

### 基本使用

```bash
# 预览视频
npx remotion studio

# 渲染视频
npx remotion render
```

## 🛠️ 技术栈

- **Remotion** — React 视频生成框架
- **TypeScript** — 类型安全
- **React** — UI 组件
